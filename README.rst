Dirsnap: Snapshot of a directory.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Use case: I want to easily upload a given directory from a Heroku dyno for
inspection/vendoring.

Usage
-----

Simple::

    $ dirsnap /app/
    Creating snapshot...
    Uploading snapshot...
    Done!
